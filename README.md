<H1>TA-RAG: Confidence-Calibrated Statement-Level Verification for Hallucination Resilient Regulatory Knowledge Systems</H1><br> 
This work proposes a Trust Aware Retrieval Augmented Generation ( TA-RAG) framework developed to mitigate hallucinations effects in regulatory question answering. The architecture Fig.1 extends a standard RAG pipeline by introducing structured evidence verification and a calibrated rejection mechanism. This ensure that generated responses remain grounded in retrieved regulatory documents. The key objective is not only to generate context aware answers but to explicitly quantify the evidential support behind each claim before presenting the final output.
